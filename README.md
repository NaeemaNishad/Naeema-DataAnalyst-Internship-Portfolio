# 🧪 Hypothesis Testing

A One-Way ANOVA test was conducted to determine whether customer spending differs significantly across age groups.

### Result

- Test: One-Way ANOVA
- F-Statistic: 0.0372
- P-Value: 0.9635

### Conclusion

The analysis failed to reject the null hypothesis, indicating that there is no statistically significant difference in average customer spending across different age groups.
